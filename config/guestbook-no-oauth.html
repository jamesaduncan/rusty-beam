Rusty Beam Guestbook Configuration (No OAuth2)

ℹ️ Testing Configuration:

This configuration runs the guestbook without Google OAuth2. Users can still post as guests.

For the full experience with Google authentication, use config/guestbook.html with proper OAuth2 credentials.

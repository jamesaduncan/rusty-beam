Default Configuration

Example server configuration for Rusty Beam. This can be used as a template for setting up your server.

Note: All plugins must be compiled as dynamic libraries (.so, .dylib, or .dll). Use ./build-plugins.sh to build the included plugins.

Property Name Property Value Property Description
Server Root ./examples/files

The root directory of the server where static files are served from.
Address 127.0.0.1

The IP address the server will bind to.
Port 3000

The port the server will listen on.
Host Name localhost

The name of the host server.
Host Root ./examples/localhost

The root directory for the host-headed server
Authentication Plugin plugins/lib/libbasic_auth.so

Plugin for user authentication
Basic Auth: Authentication File ./examples/localhost/auth/users.html
Authorization Plugin plugins/lib/libfile_authz.so

Plugin for authorization decisions
Authorization File ./examples/localhost/auth/users.html

File containing authorization rules for this host
Access Log Plugin plugins/lib/libapache_access_log.so

Plugin for access logging in Apache format
Access Log File ./logs/localhost/access.log

Path to the access log file
Error Log Plugin plugins/lib/libapache_error_log.so

Plugin for error logging in Apache format
Error Log File ./logs/localhost/error.log

Path to the error log file
Host Name example.com

The name of the host server.
Host Root ./examples/example-com

The root directory for the host-headed server

User Schema

The User schema represents an authentication user in Rusty Beam. It defines user credentials and role-based access control information.

Schema Definition

Schema URL: http://rustybeam.net/User

Properties

Property Type Cardinality Description
username Text 1 The user's login name
password Text 1 The user's password (plaintext or hashed based on encryption property)
role Text 0..n User roles for authorization (e.g., "administrators", "users")
encryption Text 0..1 Password encryption type: "plaintext" or "bcrypt" (defaults to plaintext if not specified)

ServerConfig Schema

The ServerConfig schema defines the main configuration for the Rusty Beam server, including binding address, port, and server root directory.

Schema Definition

Schema URL: http://rustybeam.net/ServerConfig

Properties

Property Type Cardinality Description
serverRoot Text 1 The root directory from which static files are served. Can be relative or absolute path
bindAddress Text 1 The IP address the server binds to (e.g., "127.0.0.1" or "0.0.0.0")
bindPort Text 1 The port number on which the server listens (e.g., "3000")

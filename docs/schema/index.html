Rusty Beam Schemas

Rusty Beam uses HTML microdata schemas for configuration and data exchange. All schemas use the http://rustybeam.net/ namespace.

Configuration Schemas

ServerConfig

Main server configuration including bind address, port, and server root.

• serverRoot
• bindAddress
• bindPort

HostConfig

Virtual host configuration for host-based routing.

• hostName
• hostRoot
• plugin[]

Plugin

Plugin configuration and loading specification.

• library
• config

Pipeline

Plugin pipeline configuration for request processing.

• plugin[]
• name

Authentication & Authorization Schemas

User

User authentication data with support for multiple encryption methods.

• username
• password
• encryption
• role[]

AuthorizationRule

Access control rules for paths and methods.

• path
• method
• require

Plugin-Specific Schemas

RedirectRule

URL redirection rules with pattern matching.

• from
• to
• permanent

RedirectRules

Collection of redirect rules.

• rule[]

StreamItem

WebSocket broadcast message format for real-time updates.

• method
• url
• selector
• content

Schema Usage Example

<div itemscope itemtype="http://rustybeam.net/User">
    <span itemprop="username">alice</span>
    <span itemprop="password">$2b$10$...</span>
    <span itemprop="encryption">bcrypt</span>
    <span itemprop="role">admin</span>
    <span itemprop="role">user</span>
</div>

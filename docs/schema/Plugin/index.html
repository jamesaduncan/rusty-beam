Plugin

Schema: http://rustybeam.net/Plugin

Property Type Cardinality Description
library https://schema.org/URL 1 The path to the plugin shared library file using file:// protocol (e.g., "file://./plugins/basic-auth.so").
plugin http://rustybeam.net/Plugin 0..n Nested plugins that form a sub-pipeline within this plugin (used for scoped plugin execution).
realm https://schema.org/Text 0..1 Authentication realm for basic-auth plugin (e.g., "Admin Area").
authfile https://schema.org/URL 0..1 Path to authentication or authorization file for basic-auth and authorization plugins.
directory https://schema.org/Text 0..1 Path prefix for directory plugin to scope nested plugins (e.g., "/admin").
logfile https://schema.org/URL 0..1 Path to log file for access-log plugin.
rulesfile https://schema.org/URL 0..1 Path to redirect rules file for redirect plugin.

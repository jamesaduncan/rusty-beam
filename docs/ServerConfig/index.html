ServerConfig

Schema: http://rustybeam.net/ServerConfig

Property Type Cardinality Description
serverRoot https://schema.org/Text 1 The root directory from which static files are served. Can be relative or absolute path.
bindAddress https://schema.org/Text 1 The IP address the server binds to (e.g., "127.0.0.1" or "0.0.0.0").
bindPort https://schema.org/Text 1 The port number on which the server listens (e.g., "3000").

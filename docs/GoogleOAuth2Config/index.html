GoogleOAuth2Config Schema

Configuration schema for the Google OAuth2 authentication plugin.

Schema URI: http://rustybeam.net/GoogleOAuth2Config
Used By: Google OAuth2 Plugin

Properties

Property Name Type Cardinality Description
client_id String 1 Google OAuth2 client ID obtained from Google Cloud Console
client_secret String 1 Google OAuth2 client secret obtained from Google Cloud Console
redirect_uri String 0..1 OAuth2 callback URL that must match the authorized redirect URI in Google Console. Defaults to http://localhost:3000/auth/google/callback
name String 0..1 Plugin instance name. Defaults to "google-oauth2"

Usage Notes

  • The client_id and client_secret are required and must be obtained from the Google Cloud Console
  • The redirect_uri must exactly match one of the authorized redirect URIs configured in Google Cloud Console
  • For local development, the default redirect URI is usually sufficient
  • For production deployments, always use HTTPS URLs for the redirect URI
  • The plugin name can be customized if running multiple instances

Security Considerations

  • Never commit the client_secret to version control
  • Consider using environment variables or secure configuration management for credentials
  • Always use HTTPS in production to protect OAuth2 tokens and session cookies
  • Regularly rotate client secrets as a security best practice

Default Configuration

Example server configuration for Rusty Beam. This can be used as a template for setting up your server.

Note: All plugins must be compiled as dynamic libraries (.so, .dylib, or .dll). Use ./build-plugins.sh to build the included plugins.

Property Name Property Value Property Description
Server Root ./files

The root directory of the server where static files are served from.
Address 127.0.0.1

The IP address the server will bind to.
Port 3000

The port the server will listen on.
Host Name localhost

The name of the host server.
Host Root ./localhost

The root directory for the host-headed server
Plugin plugins/lib/libbasic_auth.so
Basic Auth: Authentication File ./localhost/auth/users.html
Authorization File ./localhost/auth/users.html

File containing authorization rules for this host (optional - falls back to authFile if not specified)
Host Name example.com

The name of the host server.
Host Root ./example-com

The root directory for the host-headed server
